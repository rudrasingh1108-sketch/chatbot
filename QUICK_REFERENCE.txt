🔌 API ENDPOINTS (For Web Developers)
═══════════════════════════════════════════════════════════════════════════════

Chat:
   POST   /api/chat                 Send message
   GET    /api/chat-history         Get conversation history

Users:
   GET    /api/voice-profiles       List all users
   POST   /api/voice-profiles/create  Create user
   DELETE /api/voice-profiles/<name>  Delete user
   POST   /api/set-current-user     Switch user
   GET    /api/current-user         Get active user

Features:
   GET    /api/weather?city=Name    Get weather
   GET    /api/define?word=Word     Get definition
   GET    /api/joke                 Get joke
   GET    /api/news?category=general Get news
   POST   /api/calculate            Do math
   GET    /api/health               API status

🐛 COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════════

Issue                             Fix
────────────────────────────────────────────────────────────────────────────────
"Microphone not detected"        Check system audio settings
                                  Plug in microphone if needed

"Module not found errors"         Dependencies already installed
                                  Run python main.py or app.py again

"Port 5000 already in use"        Close other Flask apps
                                  Restart computer

"Speech not recognized"           Speak louder, slower, clearer
                                  Reduce background noise
                                  Check microphone volume

"Web page won't load"             Ensure app.py is running
                                  Check http://localhost:5000
                                  Restart Flask server

"Voice profile not working"       Train voice first: "Train voice"
                                  Use microphone with good quality
                                  Test with web app if needed

📈 VOICE PROFILE TRAINING
═══════════════════════════════════════════════════════════════════════════════

To train voice recognition:

1. Start voice app:           python main.py
2. Listen for wake word       App waits for "Hey Assistant"
3. Say "Hey Assistant"        Bot detects wake word
4. Say "Train voice"          Training mode starts
5. Record 3 samples           Say each sample when prompted
6. Provide username           Give your name
7. Profile saved!             Bot confirms save

Result:
   ✓ Bot recognizes you
   ✓ You get personalized greetings
   ✓ Different users get different treatment
   ✓ Profile stored in voice_profiles.json

🔐 PRIVACY & SECURITY
═══════════════════════════════════════════════════════════════════════════════

What's NOT stored:
   ✗ Microphone audio files (processed in-memory only)
   ✗ Chat messages (unless saved by user)
   ✗ API keys (use environment variables in production)

What's stored locally:
   ✓ Voice profiles (voice_profiles.json)
   ✓ Chat history (in-memory during session)
   ✓ User preferences (in-memory)

For production, add:
   → HTTPS instead of HTTP
   → User authentication
   → Database encryption
   → Rate limiting
