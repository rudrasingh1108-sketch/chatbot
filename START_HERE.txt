OPTION A: VOICE ASSISTANT (Traditional - Most Interactive)
   ────────────────────────────────────────────────────────
   Step 1: Double-click "launch.bat"  (Windows)
           OR type: python main.py    (Any OS)
   
   Step 2: Wait for "Listening for wake word..."

           "What's the weather?"
           "Tell me a joke"
           "Search for Python tutorials"
   
   Step 5: Say "Bye" to stop

OPTION B: WEB APPLICATION (Browser - Easy & Modern)
   ──────────────────────────────────────────────────
   Step 1: Double-click "launch_web.bat"  (Windows)
           OR type: python app.py         (Any OS)
   
   Step 2: Wait for "Running on http://localhost:5000"
   
   Step 3: Browser opens automatically (or go to http://localhost:5000)
   
   Step 4: Type your message or click quick actions:
           Click Weather to get weather
           Click Joke to see a funny joke
           Click News to see headlines
   
   Step 5: Close browser tab to stop

OPTION C: BOTH RUNNING (Maximum Features)
   ──────────────────────────────────────
   Step 1: Open Terminal 1 → python main.py
   Step 2: Open Terminal 2 → python app.py
   Step 3: Use voice when available, web when not
   Step 4: They share the same profiles and context!

📞 TROUBLESHOOTING (Most Common Issues)
════════════════════════════════════════════════════════════════════════════════

"Microphone not detected"
   → Check Windows Settings > Sound
   → Make sure microphone is plugged in and enabled
   → Test microphone with Audacity or settings app

"Port 5000 already in use" (Web app won't start)
   → Close other Flask applications
   → Restart your computer
   → Or kill the process using port 5000

"Speech not recognized"
   → Speak louder and slower
   → Reduce background noise
   → Speak at normal distance (not too far)
   → Check microphone quality/battery if wireless

"Python not found" error
   → Make sure Python is installed
   → Check if .venv folder exists
   → Run: python main.py (from project folder)

"Import errors"
   → Dependencies were already installed
   → Try running again
   → Or reinstall manually: pip install -r requirements.txt

For detailed troubleshooting, see:
   → README.md (Troubleshooting section)
   → WEB_APP_GUIDE.md (Common issues)
   → LAUNCH_GUIDE.txt (Quick fixes)
